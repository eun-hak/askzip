# 스크린샷 촬영·삽입 가이드

안내글의 E-E-A-T(경험) 보강을 위한 실제 수행 화면 캡처 가이드입니다.
**글별 촬영 목록은 각 `src/data/articles/<slug>.ts` 파일 하단의 `TODO(스크린샷)` 주석에 있습니다.**

## 1. 촬영 원칙

- **직접 수행하며 캡처**: 실제 발급·신청·조회를 진행하면서 단계별 화면을 캡처합니다. 검색해서 주운 이미지·타 블로그 이미지는 절대 사용 금지 (저작권 + 독창성 훼손).
- **글당 2~4장**: 모든 스텝이 아니라 "글로 설명하기 어려운 화면"만. 특히 ① 메뉴 진입점 ② 옵션 선택 화면 ③ 완료/결과 화면.
- **PC 화면 기준**, 모바일 앱 절차 글은 모바일 캡처. 브라우저 주소창을 포함해 공식 사이트임이 보이게.

## 2. 개인정보 마스킹 (필수)

캡처에 다음이 보이면 반드시 가리고 저장:

| 항목 | 처리 |
|------|------|
| 이름·주민등록번호 | 완전 블러 또는 검은 박스 |
| 주소·전화번호·계좌 | 완전 블러 |
| 인증서 관련 정보 | 완전 블러 |
| 문서확인번호·바코드 | 블러 (진위확인 악용 방지) |

- macOS 기본 미리보기(마크업 도구)로 충분. 반투명 블러보다 **불투명 도형**이 안전.
- 마스킹 후 원본은 삭제 (원본 보관 자체가 유출 리스크).

## 3. 파일 규약

- 저장 위치: `public/images/articles/<slug>/` (예: `public/images/articles/resident-registration-copy/`)
- 파일명: `01-search.png`, `02-options.png` 처럼 `순번-내용` 형식
- 포맷: PNG 또는 WebP, 가로 1200~1600px, 파일당 500KB 이하 권장 (미리보기 → 내보내기로 축소)

## 4. 글에 삽입하기

해당 글 `.ts` 파일의 `content` 배열에서 관련 단계 뒤에 `image` 블록 추가:

```ts
{
  type: 'image',
  src: '/images/articles/resident-registration-copy/02-options.png',
  alt: '정부24 주민등록등본 발급 옵션 선택 화면',
  caption: '뒷자리 표시·세대원 포함 여부를 여기서 선택합니다 (2026년 7월 기준 화면)',
  width: 1600,   // 실제 픽셀 크기
  height: 1000,
},
```

- `alt`: 화면 내용을 문장으로 (SEO + 접근성)
- `caption`: 캡처 시점 명시 권장 ("2026년 7월 기준 화면") — 화면 개편 대비
- 삽입 후 해당 글의 `TODO(스크린샷)` 주석에서 완료 항목 삭제, `updatedAt` 갱신

## 5. 현재 상태 (2026-07-06)

**인증 전 공개 화면 7장은 확보 완료** (헤드리스 브라우저로 실제 사이트 캡처, 개인정보 없음):
위택스·홈택스·전자가족관계등록시스템·국민건강보험·인터넷등기소·스마트초이스·안전운전통합민원 첫 화면 → 해당 글 7개에 삽입됨.

**남은 것은 본인인증 이후 화면** — 이건 운영자만 캡처할 수 있고, 심사 관점에서도 가장 가치 있는 부분입니다:
- 발급 옵션 선택 화면 (마스킹 후)
- 발급 완료·PDF 저장 화면 (마스킹 후)
- 각 글 하단 `TODO(스크린샷)` 주석의 나머지 항목

참고: 정부24(gov.kr)는 자동화 접속을 차단하므로 정부24 화면은 전부 직접 캡처해야 합니다.

## 6. 촬영 세션 플랜 (운영자용 — 한 번 로그인할 때 몰아서 찍기)

캡처는 Mac 기준 `Cmd+Shift+4`(영역 선택). 브라우저 창을 화면에 꽉 채우고 주소창이 보이게. 개인정보는 미리보기 마크업의 **불투명 사각형**으로 가린 뒤 저장.

### 세션 1 — 정부24 (지금 가능, 최우선: 3개 글 커버)
로그인 1번으로 등본·초본·인감 3건 발급하며 촬영:
1. "주민등록등본" 검색 결과 화면 (등본·초본 메뉴가 나란히 보이게) → 등본·초본 글 공용
2. 등본 발급 옵션 화면 (뒷자리 표시·세대원·변동사유 선택 UI 전체)
3. 초본 발급 옵션 중 "주소 변동 이력 범위" 선택 부분
4. "인감증명서" 검색 결과 + 용도(일반용) 선택 화면
5. (가능하면) 정부24 앱에서 수령 방법 "전자문서지갑" 선택 화면 (모바일 캡처)

### 세션 2 — 홈택스 (지금 가능)
1. 로그인 후 [장려금·연말정산·전자기부금] 메뉴가 펼쳐진 화면
2. 근로·자녀장려금 "심사 진행 상황 조회" 화면 (내역 마스킹)
3. [연말정산 간소화] 메뉴 진입 화면 (시즌 화면은 1~2월에 추가 촬영)

### 세션 3 — 국민건강보험 (지금 가능)
1. 민원여기요에서 "자격득실확인서" 검색 결과
2. 본인인증 수단 선택 화면 (간편인증 목록)
3. 발급된 확인서의 가입자 구분·취득일·상실일 부분 (이름·주민번호 마스킹)
4. (가능하면) The건강보험 앱 증명서 발급 메뉴 (모바일)

### 세션 4 — 전자가족관계등록시스템 (지금 가능)
1. 증명서 종류 선택 후 일반·상세·특정 표시 범위 선택 화면
2. 발급 완료 화면 (문서 내용 마스킹)

### 세션 5 — 모의계산기류 (로그인 불필요, 지금 가능)
1. 고용24 실업급여 모의계산기 입력 화면
2. 복지로 청년월세지원 모의계산 입력 화면

### 시즌 대기 (해당 시기에만 화면이 열림)
- **위택스 자동차세 연납 신청·세액 확인 화면 → 매년 1월** (1월 16일~ 신청 기간에 촬영)
- **근로장려금 신청 화면 → 5월(정기)·9월(반기)**
- **연말정산 간소화 자료 조회 화면 → 1~2월**

### 전달 방법
마스킹한 PNG를 아무 폴더에 모아 두고 "어느 글의 몇 번 항목인지"만 알려 주면, 리사이즈·파일명 정리·글 삽입·캡션 작성은 Claude가 처리합니다.

## 7. 우선순위

배치 1(핵심 10개 글)부터. 한 번에 몰아서 하지 말고 **글 2~3개씩** 진행하면서 해당 글 배포와 함께 반영하는 것이 자연스럽습니다. 촬영하며 실제로 막힌 지점이 있으면 그대로 `TODO(경험담)` 슬롯에 1~2문장으로 적어 두세요 — 그게 가장 좋은 콘텐츠입니다.
