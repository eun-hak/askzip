## 2. 참고글에서 새로 추가된 리스크 (기존 진단에 없던 것)

### 2.1 ★★★ Scaled Content Abuse (확장된 콘텐츠 악용)

Google이 2024-03부터 공식 도입한 스팸 정책. **"검색 순위를 노리고 대량 생성된, 사용자에게 부가가치 없는 페이지"는 사이트 전체가 제재** 대상.

- 애스크집은 40개 글을 하루(6/16)에 일괄 생성 → **이 정책의 전형적 패턴에 해당**
- 대응: 글을 한꺼번에 추가하지 말 것. 재작성·신규 글은 **주 2~3편씩 시간차 발행**
- AI 초안 자체가 금지는 아님. 금지는 "검토·경험·부가가치 없는 대량 생성". 글마다 직접 발급 경험·스크린샷·실패담을 넣어 "사람이 검수·보강한 콘텐츠"로 만들 것

### 2.2 ★★☆ 도메인 신선도 (Domain Freshness)

- 참고글 저자의 4회 거절 경험 중 하나가 "신규 도메인 즉시 신청" (거절률 80% 주장 — 비공식)
- 애스크집은 생성 3주 만에 신청 → 같은 패턴
- 대응: **최소 2~3개월(참고글 기준 6개월) 운영 이력 + 발행 이력 + 트래픽**을 쌓은 후 재신청

### 2.3 ★★☆ 반복 거절 리스크

- 거절 후 최소 2주 대기는 형식 요건일 뿐, **개선 없이 재신청하면 거절 이력만 누적**되고 이후 심사에 불리 (참고글: 블랙리스트 위험 주장)
- 대응: 아래 3단계 로드맵을 완료하기 전에는 재신청하지 않는다

### 2.4 ★☆☆ 승인 후 무작위 재심사 (2026 도입 주장)

- 승인이 끝이 아님. 승인 후에도 저품질 판정 시 광고 게재 중단 가능
- 대응: 승인용 벼락치기가 아니라 지속 가능한 콘텐츠 제작 프로세스를 만들 것

### 2.5 ★☆☆ 필수 페이지 5종 (기존 3종 → 5종)

- 현재 없음: **이용약관(/terms)**, **독립 면책조항(/disclaimer)** (면책 문구는 about·글 하단에 산재)
- 대응: 두 페이지 신설 + 푸터에 5종 모두 노출

## 3. 통합 실행 로드맵

### Phase 1 — 즉시 (1주 내, 코드 작업)

- [ ] `/terms` 이용약관 페이지 신설
- [ ] `/disclaimer` 면책조항 독립 페이지 신설 (글 하단 산재 문구를 통합·링크)
- [ ] Footer에 5종 페이지 링크 모두 배치
- [ ] `publishedAt` 소급 날짜를 실제 발행일로 정정 (JSON-LD·sitemap의 날짜 모순 제거)
- [ ] [about/page.tsx](../src/app/about/page.tsx)의 "스크린샷이나 첨부 이미지 없이 텍스트만으로" 문구 삭제 → 실제 캡처 기반 안내로 교체
- [ ] Article 타입에 저자(author)·이미지(image 블록) 지원 추가
- [ ] 저자 소개 페이지(또는 about 내 프로필 섹션): 필명/실명, 배경, 연락처 — 글마다 저자 표기 연결

### Phase 2 — 콘텐츠 재건 (4~8주, 핵심)

- [ ] **핵심 15개 글 선정 → 직접 발급/신청 수행 → 단계별 스크린샷 캡처(개인정보 마스킹) 삽입**
- [ ] 같은 글에 실경험 서술 추가: 막힌 지점, 소요 시간, 실패담, 제출처별 후기 → **글당 3,000자 이상** 목표 (패딩 금지, 실질 내용으로)
- [ ] 분량 편차를 자연스럽게: 깊은 글 4,000자+, 가벼운 글 2,000자 — 균일 밴드 탈피
- [ ] 나머지 얇은 글 25개: 보강 전까지 **noindex 또는 비공개** 검토 (저품질 페이지가 사이트 전체 평가를 끌어내림)
- [ ] 인터랙티브 도구 1~2개 구축: 자동차세 연납 할인 계산기, 근로장려금 예상액 모의계산기 → "요약 블로그"에서 "유틸리티 사이트"로 포지션 전환
- [ ] 신규·재작성 글은 **주 2~3편씩 시간차 발행** (일괄 업로드 금지 — Scaled Content Abuse 신호 차단)

### Phase 3 — 운영 이력 축적 (재신청 전 4~8주)

- [ ] Search Console 등록·색인 요청, 커버리지 정상 확인
- [ ] 일 자연 유입 30명 이상 달성 (참고글 기준; 최소한 "유입 0"은 벗어날 것)
- [ ] 발행 이력 그래프가 "꾸준한 운영"으로 보이는지 확인 (최소 6~8주 연속)
- [ ] 위 전부 완료 후 재신청 — **개선 미완료 상태의 조기 재신청 금지**

## 4. 글 1편 품질 기준 (재작성·신규 공통)

기존 `adsense-content-originality.md` 기준에 아래를 **추가**:

- [ ] 실제 수행 스크린샷 2장 이상 (마스킹 처리)
- [ ] 1인칭 실경험 문장 포함 ("직접 해보니 ~에서 막혔다", 소요 시간, 실패 원인)
- [ ] 본문 3,000자 내외 (실질 내용 기준, 패딩 금지)
- [ ] 저자 표기
- [ ] AI 초안 사용 시: 사실 검증 + 경험 추가 + 문장 재작성 후 게시 (원문 그대로 게시 금지)
- [ ] 발행일 = 실제 게시일 (소급·분산 조작 금지)
